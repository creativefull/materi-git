Apa itu Git ?

Git adalah sebuah tool untuk kolaborasi code kita dalam bekerja team







Mengapa Harus Pakai Git ?

Untuk mempermudah seorang programmer dalam berbagi code dalam satu team. dan juga untuk mengontrol versi source code yang sedang di kerjakan



GIT AWESOME

Basic Command

Perintah - Perintah dasar untuk menggunakan git

  • Git Init
    Inisialisasi project
  • Git Clone
    Ambil project dari respository
  • Git Remote
    Buat remote git ke server
  • Git add
    Menambah File untuk di upload
  • Git commit
    Komentar / Deskripsi perubahan kode
  • Git pull
    Ambil perubahan source code dari server
  • Git push
    Upload perubahan kode ke server
